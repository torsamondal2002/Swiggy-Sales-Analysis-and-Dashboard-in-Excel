# 📊 Swiggy Sales Analysis Dashboard

An end-to-end sales analytics project built entirely in **Microsoft Excel** to analyze Swiggy sales data and convert raw transactional records into meaningful business insights.

The project demonstrates how Excel can be used as a Business Intelligence tool by combining **Power Query**, **Pivot Tables**, **Pivot Charts**, and interactive **Slicers** to build a dynamic dashboard for business reporting.

---

## 📌 Project Overview

The objective of this project is to analyze Swiggy sales data, identify business trends, and create an interactive dashboard that helps stakeholders monitor sales performance across restaurants, cities, and states.

The dataset contains over **197,000 sales records** from **993 restaurants** across **28 Indian states** and **28 cities**, making it a realistic business analytics use case. :contentReference[oaicite:1]{index=1}

## 💡 Key Insights

Some interesting observations from the analysis include:

- Sales performance differs significantly across cities.
- Restaurants with higher customer ratings generally perform better.
- Certain food categories contribute a larger share of overall sales.
- Premium-priced dishes often receive stronger customer ratings.
- Regional analysis helps identify high-performing markets for business expansion.
